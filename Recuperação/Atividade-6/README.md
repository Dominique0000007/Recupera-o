# Atividade 6 - App Completo React Native

## 📱 Descrição
Aplicativo completo React Native com navegação por abas, múltiplas telas e funcionalidades avançadas. Inclui gerenciamento de tarefas, dashboard com estatísticas e perfil do usuário.

## 🚀 Como executar

### Pré-requisitos
- Node.js instalado
- Expo CLI: `npm install -g @expo/cli`
- App Expo Go no celular (iOS/Android)

### Executar o projeto
```bash
npm install
npm start
# ou
expo start
```

### Executar em dispositivos
```bash
expo start --android  # Android
expo start --ios     # iOS
expo start --web      # Web
```

## 🛠️ Tecnologias
- **Framework**: React Native
- **Plataforma**: Expo
