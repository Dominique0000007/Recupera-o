# Atividade 4 - Clone do Netflix

## 🎬 Descrição
Interface moderna inspirada no Netflix, desenvolvida com React. Inclui header responsivo, seção hero, e carrosséis de filmes/séries.

## 🚀 Como executar
```bash
npm install
npm start
```

## 🎨 Funcionalidades
- ✅ Header fixo com logo Netflix
- ✅ Navegação responsiva
- ✅ Seção hero com filme em destaque
- ✅ Carrosséis de filmes/séries
- ✅ Efeitos hover nos posters
- ✅ Design responsivo
- ✅ Gradientes e animações

## 🛠️ Tecnologias
- **Frontend**: React 18
- **Estilização**: CSS3 com gradientes
- **Ícones**: Emojis nativos
- **Responsividade**: Mobile-first

## 📱 Layout
- Header fixo com navegação
- Hero section com filme em destaque
- Múltiplas linhas de conteúdo
- Scroll horizontal nos carrosséis
- Design dark theme

## 🎯 Características Visuais
- Background escuro (#141414)
- Logo Netflix oficial
- Botões de reprodução e informações
- Efeitos de hover e transições
- Gradientes para fade effects
