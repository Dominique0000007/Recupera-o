# Atividade 5 - App React Native + Expo

## 📱 Descrição
Aplicativo móvel de gerenciamento de tarefas desenvolvido com React Native e Expo. Interface moderna com gradientes e animações.

## 🚀 Como executar

### Pré-requisitos
- Node.js instalado
- Expo CLI: `npm install -g @expo/cli`
- App Expo Go no celular (iOS/Android)

### Executar o projeto
```bash
npm install
npm start
# ou
expo start
```

### Executar em dispositivos
```bash
expo start --android  # Android
expo start --ios     # iOS
expo start --web      # Web
```

## 📱 Funcionalidades
- ✅ Adicionar novas tarefas
- ✅ Marcar tarefas como concluídas
- ✅ Deletar tarefas com confirmação
- ✅ Estatísticas em tempo real
- ✅ Interface responsiva
- ✅ Gradientes e animações
- ✅ Ícones nativos do Expo

## 🛠️ Tecnologias
- **Framework**: React Native
- **Plataforma**: Expo
- **Ícones**: @expo/vector-icons
- **Gradientes**: expo-linear-gradient
- **Navegação**: React Navigation (se necessário)
