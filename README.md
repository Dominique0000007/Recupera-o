## 📁 Estrutura Criada

### **Atividade-2** ✅
- **Frontend React** com integração à API externa
- **Funcionalidades**: Listar, adicionar, marcar e deletar tarefas
- **API**: jsonplaceholder.cypress.io
- **Status**: Pronta para entrega

### **Atividade-3** ✅ (Nova - Backend + Frontend)
- **Backend**: Node.js/Express com API REST completa
- **Frontend**: React separado conectado ao backend local
- **Funcionalidades**: CRUD completo com tratamento de erros
- **Status**: Pronta para entrega

### **Atividade-4** ✅ (Clone do Netflix)
- **Frontend**: React com interface moderna
- **Funcionalidades**: Header responsivo, hero section, carrosséis
- **Design**: Dark theme inspirado no Netflix
- **Status**: Pronta para entrega

### **Atividade-5** ✅ (React Native + Expo)
- **Mobile**: App nativo com Expo
- **Funcionalidades**: Gerenciamento de tarefas móvel
- **Design**: Gradientes e animações
- **Status**: Pronta para entrega

### **Atividade-6** ✅ (App Completo React Native)
- **Mobile**: App completo com navegação
- **Funcionalidades**: 3 telas (Tarefas, Dashboard, Perfil)
- **Navegação**: Bottom tabs com React Navigation
- **Status**: Pronta para entrega

## 🚀 Como executar cada atividade:

1. **Atividade-2**: `npm install && npm start`
2. **Atividade-3**: 
   - Backend: `cd backend && npm install && npm start`
   - Frontend: `cd frontend && npm install && npm start`
3. **Atividade-4**: `npm install && npm start`
4. **Atividade-5**: `npm install && expo start`
5. **Atividade-6**: `npm install && expo start`

Todas as atividades estão **completas e separadas** conforme solicitado! Cada uma tem seu próprio README com instruções detalhadas. 🎉
